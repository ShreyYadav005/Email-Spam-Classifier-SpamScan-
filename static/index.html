// How it works

1. TF-IDF Vectorization Converts email text into numerical features, weighting rare but important words higher.
2. Naive Bayes Classifier Applies Bayes' theorem assuming word independence to compute spam/ham probabilities.
3. Bigram Features Captures word pairs like "free money" for better context understanding.
4. Confidence Score The model outputs a probability, not just a label — shown as a percentage above.
